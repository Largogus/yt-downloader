## Возможности

* Скачивание видео по ссылке
* Выбор качества видео
* Автоматический выбор лучшего доступного качества
* Объединение видео и аудио через FFmpeg
* Поддержка прокси (Пока-что только встроенный)
* Удобный графический интерфейс

Из независих проблем, это ограничения работы Youtube в России, из-за чего приходится использовать proxy который может отвалиться
Возможные проблемы:
* Поломка proxy
* Слишком долгое ожидание (тоже связано с proxy)
* Другие ранее не обнаруженные проблемы
